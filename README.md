# # Log Catcher Desktop

# Log Catcher Desktop は、JSON ファイルを読み込んで TODO やメモを電光掲示板風のティッカー表示で流せる、Electron 製のデスクトップガジェットです。

Chrome 拡張 **Log Catcher** などで保存・書き出した JSON を、デスクトップ上で常時見える小さなウィジェットとして表示し、ブラウザを開かなくてもタスクやメモを確認できるようにすることを目的に作成しました。

---

## 概要

Log Catcher Desktop は、ローカルの JSON ファイルを監視し、その内容をコンパクトなデスクトップガジェットとして表示するアプリです。

単なる一覧表示ではなく、TODO やメモを横スクロールのティッカーとして流すことで、

- 今やるべきもの
- 完了済みのもの
- 覚えておきたいメモ

を、作業中でも視界の端で軽く確認できるようにしています。

---

## 主な機能

- Electron 製のデスクトップ常駐ガジェット
- JSON ファイルの読み込み
- JSON ファイルの変更監視と自動再読込
- TODO / メモのティッカー表示
- active / done / メモの表示切り替え
- 表示速度の切り替え（Slow / Normal / Fast）
- 常に手前表示の切り替え
- トレイ常駐
- 右クリックメニューからの各種操作
- 件数表示（TODO / DONE / メモ）
- ステータスランプ表示
- URL を持つ項目の外部ブラウザ起動
- ウィンドウサイズや設定の保存

---

## 想定している使い方

- Log Catcher で保存した TODO / メモをデスクトップで流す
- 作業中にブラウザを開かず、今のタスクを確認する
- 軽量な個人用タスクガジェットとして使う
- ローカル JSON を使った簡易ステータスモニターとして使う
- 小規模な運用補助ツール・自作ワークフローのフロントエンドとして使う

---

## このアプリを作った理由

ブラウザ拡張でメモや TODO を保存できても、  
「わざわざポップアップを開かないと見ない」  
「存在を忘れてしまう」  
ということがよくあります。

そこで、保存した内容をもっと自然に視界に入れられるよう、  
**デスクトップに置ける小さなガジェットとして見える化する**  
ことを目的に Log Catcher Desktop を作成しました。

大きなタスク管理アプリではなく、  
**軽量で、邪魔にならず、でも常に少し見えている**  
という距離感を大切にしています。

---

## 動作イメージ

1. Log Catcher などで JSON を作成・保存
2. Log Catcher Desktop で JSON ファイルを選択
3. JSON を読み込んで TODO / メモをティッカー表示
4. JSON が更新されると自動で再反映
5. トレイメニューから表示条件や速度を変更

---

## 表示ルール

JSON から読み込んだ項目は、以下の優先順で並びます。

1. active の TODO
2. メモ
3. done の TODO

これにより、今やるべきものを最初に見せつつ、補足メモや完了済み項目も必要に応じて流せるようにしています。

---

## 主な設定項目

- JSON ファイルの選択
- 今すぐ再読込
- active のみ流す
- done も含める
- メモも含める
- 表示速度の変更
- 常に手前表示

設定内容はローカルに保存され、次回起動時にも引き継がれます。

## 技術構成

- Electron
- JavaScript
- HTML / CSS

---

## ファイル構成

### `main.js`
アプリ全体の制御を担当します。

主な役割:

- アプリ起動
- ウィンドウ作成
- トレイ作成
- 右クリックメニュー構築
- JSON ファイル選択
- JSON 読込
- ファイル監視
- 設定保存
- renderer へのデータ送信
- URL の外部ブラウザ起動

### `renderer.js`
表示とアニメーションを担当します。

主な役割:

- TODO / メモの表示内容生成
- ティッカーの横スクロール表示
- ホバー時の停止制御
- 表示件数・位置情報の更新
- ステータスランプのアニメーション
- エラー表示 / ヒント表示
- クリック時の URL 起動

### `styles.css`
ガジェットUIの見た目を定義しています。

主な要素:

- 黒ベースのガジェット外観
- ランプ表示
- ティッカー枠
- メタ情報パネル
- エラーバー / ヒントバー
